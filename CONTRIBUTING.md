# Contributing to ByteBerry

Thank you for considering contributing to **ByteBerry OAuth2 Service** 🎉

## 🚀 How to Contribute
1. Fork the repository
2. Create a new branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -m 'feat: add new feature'`)
4. Push to the branch (`git push origin feature/your-feature`)
5. Open a Pull Request

## ✅ Guidelines
- Follow [Conventional Commits](https://www.conventionalcommits.org/) (e.g., `feat:`, `fix:`, `docs:`)
- Ensure CI/CD passes before requesting review
- Add/update tests if applicable
- Keep PRs focused and small
